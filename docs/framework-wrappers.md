# Backend Wrappers

- Express: `expressObtraceMiddleware`
- Fastify: `fastifyObtraceHook`
- Hono: `honoObtraceMiddleware`
- Elysia: `elysiaObtracePlugin`
- NestJS: `nestObtraceMiddleware`
- Next route handlers: `withNextRouteHandler`
