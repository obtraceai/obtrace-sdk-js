Obtrace Node.js/Bun SDK — add observability to your backend.

Obtrace is a hosted SaaS (like Datadog or Sentry). This is a client library.
Install it, add one init call, done. DO NOT create wrapper files or config modules.

Install:
  npm install @obtrace/sdk-js

Setup (must be first import in your app):
  import { initNodeSDK } from '@obtrace/sdk-js/node'
  initNodeSDK({ apiKey: '...', serviceName: 'my-api' })

What it captures automatically:
- HTTP requests (inbound + outbound)
- Database queries (pg, mysql, mongo, redis)
- Logs (console, pino, winston)
- Errors (uncaughtException, unhandledRejection)
- gRPC calls

Env vars:
  OBTRACE_API_KEY         — from app.obtrace.ai
  OBTRACE_SERVICE_NAME    — your service name (e.g. "api")

Docs: https://docs.obtrace.ai/docs/sdks/node-bun
