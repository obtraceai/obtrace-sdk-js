Obtrace SDK JS (Backend)

Purpose:
- Backend JS SDK for telemetry, context propagation, and ingest transport.
- Runtime targets: Node.js and Bun.

Key entrypoints:
- @obtrace/sdk-js/node -> initNodeSDK
- @obtrace/sdk-js -> ObtraceClient + backend wrappers

Ingest endpoints:
- POST /otlp/v1/logs
- POST /otlp/v1/traces
- POST /otlp/v1/metrics
- POST /ingest/replay/chunk
- POST /ingest/replay/recipes

Design constraints:
- SDK is thin/dumb.
- Business logic is server-side (ingest/control-plane/backend).

Core capabilities:
- Node/Bun: logs/metrics/spans, fetch instrumentation, unhandled error hooks.
- Wrappers: Express, Fastify, Hono, Elysia, Nest, Next route handlers.
